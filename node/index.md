# 事件驱动

有点像参赛者模式。

```
// 把事件发送到事件队列中


                                         |-------->|
  |-------|                     events   |         |
  |       |   EventEmitter   |--| |--| |--|        |
  |       | ---------------> |  | |  | |  |     event loop ----> event handlers
  |       |                  |--| |--| |--|        |
  |-------|                              ^         |
                                         |---------|
```

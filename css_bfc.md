#bfc
(block formatting context 块格式化上下文)  
bfc的特性就是包裹浮动元素。  

##功能

2. 阻止内部元素逃离盒子。  
3. 阻止内部元素从盒子里伸出来。  
4. 阻止内部元素与外界联系。（自己的理解）

##如何创建bfc
`floata: xxx;`  
`position: absolute, fixed`  
`display: inline-block, table-cell, table-caption, table-cell`

2. bfc

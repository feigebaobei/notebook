# 使用时遇到的问题

## 为什么mongodb启用需要先启用mongod再启用mongo
原理：
mongod是服务端，mongo是运行命令行中的客户端。生产环境中会让服务端与客户端在一起。
同理，mysql里，mysqld是服务端，mysql是命令行客户端。

## 启动方式

```
// 1
cd <mongodbPath>/bin
mongod.exe // win
./mongod // mac

// 2
cd <mongodbPath>/bin
mongod // win

// 3
echo "MongoDB starting.........."
mongod --dbpath E:\db\MongoDB
pause
```

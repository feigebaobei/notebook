# sfile

这是七牛的命令行工具。

```
npm i -g sfile
sfile [library] // search
sfile search [library] [Options]
  -s, --ssl 输出https链接
  -h, --html 输出html链接
  -j, --jade 输出jade markup
  -n, --no-link 不显示最后的版本的链接
sfile get [library]
  -s, --ssl
  -h, --html
  -j, --jade
  -v, --version
  -c, --copy
sfile --version
```
